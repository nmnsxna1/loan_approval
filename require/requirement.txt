Build a modern, production-quality AI Loan Approval Platform using React (Vite), Tailwind CSS, Node.js, Express, PostgreSQL, and Prisma.

The application should have a professional banking/insurance dashboard UI with a collapsible left sidebar, responsive design, cards, tables, modals, loading indicators, notifications, and role-based authentication.

There are only three roles:

1. Applicant
2. Policy Manager
3. Main Manager

Use simple username/password authentication for each role.
